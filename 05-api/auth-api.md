## 目标

支持教师与学生登录、身份校验和基础资料管理。

## 接口清单

| 方法 | 路径 | 说明 | 权限 |
| --- | --- | --- | --- |
| POST | `/auth/login` | 登录 | 公开 |
| POST | `/auth/logout` | 退出登录 | 已登录 |
| GET | `/users/me` | 获取当前用户 | 已登录 |
| PUT | `/users/me` | 更新个人资料 | 已登录 |
