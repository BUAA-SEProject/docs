# 术语表

| 术语 | 说明 |
| --- | --- |
| AUBB | Academic Unified Builder Bench，平台正式名称 |
| 平台配置 | 平台名称、主题、模块开关等平台级设置 |
| 组织单元 | 院系、班级、实验班等组织结构节点 |
| 课程 | 教师组织教学活动的基本单元 |
| 课程成员 | 参与课程的教师、助教或学生 |
| 任务 | 作业、实验或报告类教学任务 |
| 提交 | 学员上传代码、文件或结果的行为 |
| 评测 | 系统对提交进行自动编译、执行和评分的过程 |
| 判题 Worker | 消费评测任务并调用 Judge0 的后台进程 |
| 批改 | 教师对提交进行人工点评和补充评分 |
| 成绩 | 根据评测和批改结果得到的最终得分 |
| 审计日志 | 对关键行为进行留痕的记录 |
