# [占位] 前端文档

## 目标

提供教师端与学生端的统一 Web 体验，重点保障主链路易用、状态清晰、演示稳定。

## 建议页面结构

- 登录页
- 课程列表页
- 课程详情页
- 作业/实验详情页
- 在线实验页
- 提交记录页
- 批改页
- 成绩统计页
- 通知中心页
- 个人中心页

## 建议前端约定

- 页面按角色分入口，但组件尽量复用
- 表单、表格、分页、筛选统一交互规范
- 统一处理加载态、空态、异常态和权限态
- 关键流程保留明确的成功/失败反馈
