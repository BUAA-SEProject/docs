# 用户手册

## 1. 文档信息

- 文档名称：AUBB（Academic Unified Builder Bench）用户手册
- 版本：v1.0
- 状态：正式基线
- 更新日期：2026-04-14
- 适用对象：平台管理员、教师、助教、学员

## 2. 面向对象与使用前提

- 推荐浏览器：Chrome、Edge、Firefox 最新两个主要版本。
- 建议分辨率：不低于 1280x720。
- 学员提交编程任务时无需安装本地开发环境，但应保证网络稳定。

## 3. 教师端操作说明

### 3.1 创建课程

1. 进入“课程管理”。
2. 点击“新建课程”。
3. 填写课程名称、学期、组织归属、简介。
4. 保存后维护成员和资源。

### 3.2 发布任务

1. 进入课程详情页的“任务管理”。
2. 点击“新建任务”，填写任务说明、开放时间、截止时间和评分方式。
3. 如为编程任务，配置支持语言和测试用例。
4. 发布任务并确认学员可见。

### 3.3 查看提交与批改

1. 进入“批改中心”或任务提交列表。
2. 按未批改、已评测、异常等条件筛选提交。
3. 查看自动评测结果后录入评语和人工评分。

### 3.4 发布成绩

1. 在任务成绩页检查成绩完整性。
2. 点击“发布成绩”。
3. 如需修改，完成成绩调整后点击“重新发布成绩”。

## 4. 学生端操作说明

### 4.1 查看课程

1. 课程成员由教师、整课助教或管理员添加。
2. 在“我的课程”查看已加入课程。
3. 点击课程进入课程详情页。

### 4.2 查看任务并提交

1. 打开课程任务详情。
2. 阅读任务说明、截止时间、支持语言和评分规则。
3. 使用在线编辑器或上传文件完成提交。
4. 系统返回提交编号后，可在“我的提交”查看状态。

### 4.3 查看评测与成绩

1. 在任务详情页查看评测状态。
2. 评测完成后查看通过情况、错误摘要和得分。
3. 成绩发布后在“成绩详情”查看总分和评语。

## 5. 管理员操作说明

### 5.1 初始化平台

1. 进入“平台配置”设置平台名称、Logo、首页文案和模块开关。
2. 保存后立即生效，无需额外发布。

### 5.2 维护组织与用户

1. 在“组织用户”中按“学校 -> 学院 -> 课程 -> 班级”维护组织结构。
2. 通过 CSV 模板批量导入教师和学员，系统会返回逐行校验结果。
3. 分配平台治理身份并检查账号状态、默认归属组织和最近登录信息。

### 5.3 查看平台概览与审计

1. 在“平台概览”查看活跃课程、活跃用户、提交量和评测成功率。
2. 在“审计日志”按时间、用户、操作类型检索关键行为。

## 6. 常见问题

| 问题 | 处理建议 |
| --- | --- |
| 登录后仍无法访问受保护页面 | 检查 JWT 是否过期、账号状态以及系统时间 |
| 提交后长时间无结果 | 检查提交状态是否仍为处理中，并联系教师或管理员 |
| 看不到成绩 | 确认教师是否已发布成绩，以及当前课程和任务是否正确 |
