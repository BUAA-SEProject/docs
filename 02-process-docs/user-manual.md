# 用户手册

## 1. 文档信息

- 文档名称：AUBB（Academic Unified Builder Bench）用户手册
- 版本：v1.1
- 状态：正式基线
- 更新日期：2026-06-12
- 适用对象：平台管理员、教师、助教、学员

## 2. 面向对象与使用前提

- 推荐浏览器：Chrome、Edge、Firefox 最新两个主要版本。
- 建议分辨率：不低于 1280x720。
- 学员完成编程作业时无需安装本地开发环境。
- 环境型实验的 Web 终端由平台提供，是否可用取决于实验运行时配置。

## 3. 管理员操作说明

### 3.1 初始化平台

1. 进入“平台配置”，维护平台名称、简称、Logo、页脚和登录提示。
2. 按“学校 -> 学院 -> 课程 -> 班级”维护组织树。
3. 创建或导入用户，并为管理员、教师、助教和学员分配合适身份或课程成员关系。

### 3.2 治理与审计

1. 在“权限解释”中选择用户、权限码和资源，检查授权原因。
2. 在“审计日志”中按时间、操作者、动作和结果检索关键行为。
3. 在“开课管理”“课程目录”“学期管理”等入口维护教学基础数据。

## 4. 教师与助教操作说明

### 4.1 维护课程

1. 进入“教学工作台”或“教学事务 -> 全部课程”，打开课程工作区。
2. 通过课程上下文入口维护教学班、成员、公告、资源、讨论、作业、提交、实验、成绩、题库和判题环境。
3. 确认教学班功能开关符合当前课程安排。

### 4.2 发布作业

1. 进入“教学事务 -> 全部作业”或课程工作区中的作业入口并创建作业。
2. 填写标题、开放时间、截止时间和提交规则。
3. 在试卷编辑中维护题目、分值、编程题语言和评测环境。
4. 检查无误后发布作业；发布后学员可在“学习中心 -> 全部作业”或课程作业入口查看。

### 4.3 查看提交与评测

1. 进入“教学事务 -> 全部提交”、课程提交入口或作业提交列表。
2. 按课程、作业、学员和评测状态筛选提交。
3. 查看分题答案、附件、自动评测报告和源码快照。
4. 对需要人工评分的题目录入分数和评语。

### 4.4 发布成绩

1. 进入“成绩册”，检查作业列、学生行、统计和未完成项。
2. 对需要调整的成绩执行人工评分、批量调整或导入。
3. 满足发布条件后发布成绩；如需修正，完成调整后重新发布。

### 4.5 管理实验

1. 进入“教学事务 -> 全部实验”或课程实验入口，创建报告型实验或环境型实验。
2. 报告型实验主要维护实验说明、附件要求和截止时间。
3. 环境型实验需要选择可用环境模板，发布后学员可启动个人会话。
4. 查看实验报告、附件和终端实验会话状态，并按需评阅或发布评语。

## 5. 学员操作说明

### 5.1 查看课程内容

1. 在“学习中心”或“全部课程”查看已加入课程。
2. 进入课程详情后查看公告、资源和讨论。
3. 只会看到自己所属课程和教学班范围内的内容。

### 5.2 完成作业

1. 在“学习中心 -> 全部作业”或课程作业入口打开作业详情。
2. 阅读题目说明、截止时间、提交要求和评分规则。
3. 对编程题进入在线工作区，编辑文件、保存、试运行并查看输出。
4. 返回作业详情后提交整份作业；提交成功后可查看提交详情。

### 5.3 查看评测与成绩

1. 在提交详情中查看评测状态、得分摘要和报告。
2. 评测失败或超时时，根据错误摘要联系教师或等待重评。
3. 成绩发布后，在“我的成绩”查看总分、反馈和可导出结果。

### 5.4 完成实验

1. 在“学习中心 -> 全部实验”或课程实验入口打开已发布实验；从全部实验进入时先确认教学班筛选范围。
2. 报告型实验可填写报告正文、上传附件并提交。
3. 环境型实验可启动个人实验环境，使用 Web 终端完成操作。
4. 教师发布评语后，可在实验详情中查看反馈。

## 6. 常见问题

| 问题 | 处理建议 |
| --- | --- |
| 登录后仍无法访问受保护页面 | 检查账号状态、课程成员关系和当前角色入口 |
| 看不到课程或作业 | 确认是否已被加入课程、教学班或作业是否已发布 |
| IDE 保存失败 | 检查网络、作业状态和是否存在工作区修订冲突 |
| 提交后长时间无评测结果 | 检查提交详情状态，必要时联系教师重排队 |
| 看不到成绩 | 确认教师是否已发布成绩，以及自己是否属于该作业范围 |
| 终端实验无法连接 | 重新申请连接 token；若仍失败，联系教师检查实验运行时 |
