# 项目章程

## 1. 项目名称

在线教学与实训平台

可在立项确认后补充正式产品名与英文名。

## 2. 项目目标

面向教师与学生，建设一个覆盖课程管理、作业/实验管理、在线实验、自动评测、批改反馈和成绩统计的一体化平台。

## 3. 目标用户

- 教师：创建课程、发布作业、批改与查看统计
- 学生：加入课程、完成实验、提交作业、查看反馈
- 平台管理员：维护基础数据、查看系统状态、处理异常

## 4. 范围定义

### 本期范围

- Web 端主流程闭环
- 核心教学功能闭环
- 课程大作业所需的全部过程文档
- 可演示、可部署、可答辩的稳定版本

### 暂不强制范围

- 完整多端适配
- 复杂商业化能力
- 超大规模高并发能力

## 5. 成功标准

- 所有核心需求均可在系统内演示并在文档中找到对应依据
- 过程文档完整且与实现一致
- 系统能在演示环境稳定运行
- 团队分工、模块边界和职责说明清晰

## 6. 文档范围

本 GitBook 站点覆盖：

- 课程要求与项目规划
- 需求、设计、测试、部署、用户手册等正式交付文档
- 前端、后端、客户端、数据库和 API 研发文档
- 团队协作规范、任务跟踪、术语与 GitBook 同步方式
